# Relay Level Generator
Automatic level generation trained using reinforcement learning. Uses A3C algorithm, OpenAI Gym, Keras and Tensorflow.

## Requirements
- Python 3.5
- Keras
- Gym

```
pip3 install keras
pip3 install gym[all]
```
